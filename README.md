# Satisfactory_saved

存档1MB~2MB大小，**位置【C:\Users\你的用户名\AppData\Local\FactoryGame\Saved\SaveGames】**，里面有个像乱码一样的文件夹（类似于6620a56ks71d7d92dd71805be9ac33），存档都在里面，这个文件夹名字是用户识别码，不可改名或删除。

上传时要复制里面的存档文件（.sav) 放放到仓库里，下载使用的时候也在上述文件夹里，把存档相应的放到文件夹里面就行了。

存档自带时间，拉取的时候选择最新时间的存档就行，**拉取的人退出的时候记得上传最新存档到仓库**（因为这时候你是服主）。
